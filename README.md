# Pywrangle
Library with auxiliary functions to clean pandas data frames. Available on PyPI [here](https://pypi.org/project/pywrangle/)

## Install
- Python 3.6+
- numpy
- pandas

To install pywrangle, use pip:
```
pip install pywrangle
```
